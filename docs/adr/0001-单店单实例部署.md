# ADR-0001：单店单实例部署

- 状态：已采纳
- 日期：2026-07-15

## 背景

产品供单个商家自行部署、选品、销售和履约，不运营一个集中式多租户平台。为尚未存在的租户隔离、平台审核和平台结算增加复杂度，会削弱首版的可部署性和可维护性。

## 决策

每套部署只服务一个商家，店铺配置是全局单例。商家拥有自己的数据库、缓存、媒体卷、域名、备份和第三方凭据。系统不存在平台管理员和跨商家数据访问；扩容优先采用单店纵向扩容或复制整套实例。

## 影响

- 权限模型只处理店主、店员和消费者，不加入 tenant id。
- 备份、恢复、监控和升级均以整套商家实例为边界。
- 若未来需要 SaaS 多租户，必须另立 ADR，并重新设计隔离、计费、审核和迁移方案。
